# Audits

The **canonical, dated audit log** is
[`docs/governance/audit-report.md`](../../docs/governance/audit-report.md).

Store ad-hoc or tool-generated audit outputs here as
`audit-<YYYY-MM-DD>-<topic>.md` (e.g. `audit-2026-06-18-lighthouse.md`), then
summarize the result into the canonical report and update
`docs/governance/{production-checklist,technical-debt,backlog,roadmap}.md`.

Use the template at
[`.claude/TEMPLATES/audit-template.md`](../../.claude/TEMPLATES/audit-template.md).
