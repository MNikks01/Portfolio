# ADR-001 — Next.js App Router portfolio stack

- **Status:** Accepted
- **Date:** 2025 (initial)

## Context

Needed a fast, SEO-friendly, animation-rich personal site that doubles as proof
of senior/product/AI engineering ability.

## Decision

Build a statically prerendered single-page site on **Next.js (App Router) +
React + TypeScript + Tailwind**, with **Framer Motion** for animation and
**three.js / react-three-fiber** for a 3D hero. Smooth scroll via **Lenis**.

## Consequences

- Excellent SEO/metadata via the App Router `metadata` API.
- Rich motion and a memorable 3D hero.
- three.js adds bundle weight → must be lazy-loaded and isolated (see ADR-002,
  and the performance backlog).
