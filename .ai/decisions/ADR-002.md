# ADR-002 — React 19 + react-three-fiber v9 alignment

- **Status:** Accepted
- **Date:** 2026-06-17

## Context

Next 15.5's App Router runs React 19 internals. `@react-three/fiber` v8 targets
React 18 and read `ReactCurrentOwner`, which React 19 renamed — crashing the 3D
hero with "Cannot read properties of undefined (reading 'ReactCurrentOwner')".

## Decision

Align the whole stack on **React 19**: upgrade `react`/`react-dom` to 19,
`@react-three/fiber` to v9, `@react-three/drei` to v10, `three` to 0.184, and the
React type packages to v19.

## Consequences

- The hero renders correctly under Next 15.5.
- Also wrapped the canvas in `CanvasBoundary` + a `webglcontextlost` handler so a
  GL failure degrades to a glow instead of crashing the page.
